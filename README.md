### <ins>Running the Website</ins>

**Backend**

Enter in the terminal (backend directory):

_try:_

    npm run dev

_else try_:

    npm i

    npm run dev

_else:_

Delete **node_modules**

    npm i

    npm run dev

**Frontend**

Enter in the terminal (M3 directory):

_try:_

    npm run dev

_else try_:

    npm i

    npm run dev

_else:_

Delete **node_modules**

    npm i

    npm run dev

*To kill a server without killing the whole terminal press _ctrl + c_
